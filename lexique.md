# Lexique Chantier — Abréviations & Termes Techniques

**Version:** 1.0.0
**Dernière mise à jour:** 2026-03-27

L'agent applique automatiquement ces conversions dans tous les journaux et rapports.
Ajouter une ligne ici = l'agent l'utilisera dès la prochaine session.

---

## Format

| Expression parlée         | Terme technique | Notes                        |
|---------------------------|-----------------|------------------------------|
| Fermeture Complète        | FC              |                              |
| Fermeture Partielle       | FP              |                              |
| Avis de Non-Conformité    | ANC             |                              |
| Ordre de Changement       | OC              |                              |
| Bon de Travail            | BT              |                              |
| Plan de Santé Sécurité    | PSS             |                              |
| Équipement de Protection  | ÉPI             |                              |

---

## Comment ajouter un terme

1. Ajouter une ligne dans le tableau
2. Incrémenter la version
3. `git commit -m "lexique: ajout [terme]"`
4. Push → l'équipe tire la mise à jour
